정글넷 계정 생성

베타 버전을 정글넷에서 테스트 가능합니다. 다음 순서대로 계정을 생성하세요.

1. 키 생성

키 생성 페이지 에서 키를 생성하세요.

2. 계정 등록

정글넷 계정 등록 페이지 에서 계정을 등록하세요. 계정명을 입력하고 오너키와 액티브 키에 1번에서 생성한 public 키를 입력합니다.

  • 기본 값을 모두 지우고 다시 입력하세요. (계정 이름을 새로 적어야 합니다!)
  • 생성을 하면 붉은색 로그가 보이는데, 정상 생성된것이니 다음단계로 진행하세요.
3. 공짜 EOS 수령

지급 페이지 에 계정을 입력하면 EOS가 지급됩니다.

  • 생성을 하면 붉은색 로그가 보이는데, 정상 지급 된 것입니다.
4. 스캐터 설치

PC Chrome 브라우저에 Scatter 플러그인을 설치합니다. (이미 설치되어 있으면 다음단계로 갑니다.) 설치가 완료되면 크럼 브라우저에 다음 아이콘이 추가된 것을 확인할 수 있습니다.

5. 스캐터 기본설정

스캐터 아이콘을 눌러 기본 설정을 합니다. (이미 설치되어 있으면 다음단계로 갑니다.)

  1. 비밀번호를 입력후 "Crate New Scatter" 버튼을 누릅니다.
  2. Mnemonic 내용을 안전한 곳에 기록해 둔 후, "I wrote it down" 버튼을 누릅니다.
  3. 정글넷 구성을 할 것이므로 "Skip Basic Setup"을 누릅니다.
6. 한국어로 언어 변경

언어를 한국어로 변경합니다.

  1. 설정 아이콘을 누릅니다.
  2. Language 메뉴를 선택합니다.
  3. 언어를 한국어로 바꾸고 "Change Languge" 버튼을 누릅니다.
7. 정글넷 추가

정글넷 네트워크를 추가합니다.

  1. 설정 메뉴에서 네트워크 메뉴를 선택합니다.
  2. 신규 생성을 누릅니다.
  3. 아래 정보를 입력하고 저장 버튼을 누릅니다.

8. 키 등록

위에서 생성한 키를 스캐터에 등록합니다.

  1. 키 쌍 메뉴를 선택합니다.
  2. 신규 생성 버튼을 누릅니다.
  3. 키를 구분하기 위해 이름을 넣은 후 "액티브 개인키"를 복사 합니다.
  4. 저장 버튼을 누릅니다.
9. 신원인증 ID 추가 #1

계정 정보를 추가합니다.

  1. 메인 메뉴에서 "신원인증 ID"를 선택합니다.
  2. 신규 생성 버튼을 누릅니다.
  3. 정글넷을 선택후 방금 8번에서 등록한 키쌍을 선택합니다.
  4. 가져오기 버튼을 누릅니다.
10. 신원인증 ID 추가 #2
  1. id@active 권한을 선택한후 "선택한 계정을 사용" 버튼을 클릭합니다.
  2. 저장 버튼을 누릅니다.
  3. 올바로 등록되었는지 확인합니다.
11. 게임으로 가기
EOS Knights 웹 버전 페이지로 갑니다.
  1. 인증창이 자동으로 표시됩니다.
  2. "신원인증 ID선택" 버튼을 누릅니다.
  3. 수락 버튼을 누릅니다.
  4. 이제 게임을 즐기실 수 있습니다.
