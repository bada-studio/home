How to crate new account

EOS Knights is a game for EOS holders. The game does not provide account creation. You can become an EOS holder by using the agency service.

Account creation services

You can create an account through the following services. EOS is required as a fee in order to create an account.

  • zeos
    https://www.zeos.co
  • eostart
    https://www.eostart.com
  • eos-account-creator
    https://eos-account-creator.com
