Create Junglenet Account

Beta version can be tested on JungleNet. Please follow these steps to create an account.

1. Generate Keys

Generate keys at this page.

2. Account Registration

Register your account on the JungleNet registration page. Enter the account name and the public keys created in step 1. Store the private key in a safe place.

  • Remove all then fill them out. (Don't forget to fill new account name.)
  • If successful, you will see a red log.
3. Get free EOS

Get free EOS on faucet page.

  • If successful, you will see a red log.
4. Install Scatter

Install Scatter plug-in on your PC Chrome browser. (If it is already installed, go to the next step.) When the installation is complete, you will see the following icon on the Chrome browser.

5. Scatter setup

Tap the scatter icon to setup. (If already installed, go to the next step.)

  1. After entering the password, click the "Create New Scatter" button.
  2. Record the contents of Mnemonic in a safe place and press the "I wrote it down" button.
  3. Press "Skip Basic Setup".
6. Add the Junglenet network

Add the jungle net network.

  1. From the "Settings" menu, select "Network" Menu.
  2. Click "New" button
  3. Enter the information below and click the "Save" button.

7. Register Key Pairs

Register the key generated on Junglenet on the scatter.

  1. Select "Key Pairs" menu
  2. Click "New" button
  3. To distinguish the keys, insert the name and copy the "Active Private Key".
  4. Click "Save" button.
8. Add Idenetity #1

Add your account infomation

  1. Select "Idenetities" in main menu
  2. Click "New" button.
  3. Select the jungle net and then select the key pair you just registered in step 7.
  4. Click the "Import" button.
9. Add Idenetity #2
  1. Select "id@active" then click the "Use Selected Account" button.
  2. Click "Save" button.
  3. Check your identity
10. Go back to the game
Go to EOS Knights web page.
  1. The Authentication window is automatically displayed.
  2. Click the "Select Identity Authentication ID" button.
  3. Click "Accept" button.
  4. That's it. Now you can enjoy the game.
